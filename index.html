JustHideMe

Easily create an ephemeral "burner" VPN.

Step 1

Create a cloud VM on Vultr, DigitalOcean, OVH, Atlantic.net, etc... (pick an image with Docker preinstalled)

Step 2

Log in and run the bootstrap:

curl -fs justhide.me | bash

Step 3

Destroy the VM when you're done.

How it all works...

The Bootstrap

JustHideMe turns the complex process of setting up a VPN server into a simple one-line shell script by way of its bootstrap process. The bootstrap is open source, and if you're not comfortable doing the 'curl | bash' thing we provide alternatives.

» https://github.com/justhideme/bootstrap

The Image

Our 'ephemeral-vpn' Docker image is the core of the JustHideMe process. Upon launching, the image configures and starts a dispostable OpenVPN server. Once it's running, the image allows you to download the necessary client configuration, either via copy-and-paste, or secure download (both URL and QRCode are provided).

Complete documentation is found here:

» https://github.com/justhideme/ephemeral-vpn
